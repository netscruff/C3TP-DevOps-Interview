What is Mesos?

  • Open-source cluster manager
  • Distributed system to run distributed applications
  • Fine-grained resource sharing and isolation
  • Sharing across distributed applications and frameworks

Mesos Arcitecture

Types of Workloads Cloud native applications Support for diverse kinds of workloads such as big data, cloud native apps etc.

Mesos frameworks

  • Long Running Services
    • Marathon - PaaS, ensures application is always on
  • Big Data Processing
    • Hadoop - MapReduce jobs
    • MPI - MPICH3 based implementation
    • Spark - general purpose cluster computing system

Mesos frameworks

  • Batch Scheduling
    • Chronos - distributed job scheduler
    • Jenkins - CI server
  • Data Storage
    • Cassandra - performant and highly available distributed database
